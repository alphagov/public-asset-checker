# Owning Public Asset Checker

The owning team for this repo is responsible for the following maintenance tasks:

- Dependency Updates (as required)
- Maintenance of the `govuk-public-asset-checker` heroku app (which is where this runs)
- Checking the slack channel that the app notifies [`#govuk-publishing-components`](https://gds.slack.com/archives/C04HLT6DY4S) and ensuring a process for dealing with alerts
